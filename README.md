# skribbl.io

## Running the project on your local machine

Move into the frontend directory:
```
cd frontend
```

Install the required packages:
```
npm i
```

Run the server locally:
```
npm run dev
```

Move into the backend directory:
```
cd backend
```

Install the required packages:
```
npm i
```

Run the server locally:
```
npm run watch
```

Visit http://localhost:3000
