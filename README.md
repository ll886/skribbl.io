# skribbl.io

## Running the project on your local machine

In one terminal, move into the frontend directory:
```
cd frontend
```

Install the required packages:
```
npm i
```

Run:
```
npm run dev
```

In another terminal, move into the backend directory:
```
cd backend
```

Install the required packages:
```
npm i
```

Set up database
```
npm run setup
```

Run:
```
npm run watch
```

Visit http://localhost:3000
